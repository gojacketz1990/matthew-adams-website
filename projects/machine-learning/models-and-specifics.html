What Are Machine Learning Models?

A machine learning model is esssentially a computer program that has been trained to recognize patterns, make decisioins or generate predictions from data. Basically it is a computer with a brain that learns from experience, much like you or I would.

Types of Supervised Learning Tasks

When dealing with Supervised Learning there are two fundamental tasks that you will find:

  • ClassificationClassification is a supervised learning task where the goal is to predict a discrete, categorical label or class given the input.
    • Binary Classification - two possible outputs (e.g. span/not spam, yes/no, malignant/benign). This is the simplest form.
    • Multi-class Classificaiton - more than two output categories but each belongs to only one category (e.g. cat, dog, bird)
    • Multi-label Classification - An input can belong to multiple categories simultaneously (e.g., an image containing both a "cat" and a "dog").

    Common Applications

    • Email Spam Detection
    • Image Recognition
    • Medical Diagnosis
    • Sentimental Analysis
    • Fraud Detection
  • RegressionRegression is a supervised learning task where the goal is to predict a continuous numerical output for a given input. Imagine drawing a line or curve through data points to estimate future values.
    • Continuous Output - The output variable can take any value within a range (e.g., real numbers). There are infinitely many possible outputs.

    Common Applications

    • House Price Prediciton
    • Stock Price Prediciton
    • Temperature Prediciton
    • Sales Forecasting
    • Age Prediction
    • Credit Score Prediction

Types of Machine Learning Models

For the most part we will stick to Supervised Learning Models, but here is some info on other types as well.

  • Supervised Learning ModelsThese models are trained on labeled data which means that the input data is paired with correct output or "answer". The model learns to map inputs to desired outputs.
    • Common Types
      • Classification
      • Regression
    • Algorithm Examples
      • Linear Regression
      • Logistic Regression
      • Decision Trees
      • Support Vector Machines
      • K-Nearest Neighbors
      • Neural Networks
  • Unupervised Learning ModelsThese models are given unlabeled data and are tasked with finding hidden patterns, structures, or relationships within the data on their own, without specific guidance.
    • Common Types
      • Clustering
      • Dimensionality Reduction
      • Association
    • Algorithm Examples
      • K-Means Clustering
      • Hierarchical Clustering
      • Principal Component Analysis
  • Reinforcement Learning ModelsThese models learn by interacting with an environment through trial and error. They receive rewards for desired actions and penalties for undesirable ones, aiming to maximize their cumulative reward over time.
  • Semi-supervised Learning ModelsThese models combine elements of both supervised and unsupervised learning. They leverage a small amount of labeled data along with a larger amount of unlabeled data to improve learning, especially when labeling data is costly or time-consuming.
  • Deep Learning Learning ModelsA subset of machine learning, deep learning models are characterized by their "deep" artificial neural networks, which have multiple layers. These models can learn complex representations from raw, unstructured data (like images, audio, or text) and are capable of more human-like functions.
